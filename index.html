Grive.

Grive is a CLI (Command line interface) for Google Drive, it's written in Go. To build it you need Go lang along with the dependencies specified in the read me.

Usage is incredibly simple. grive upload [title] [filename] to upload file
grive remove [title] to remove file grive rename [target_file_name] [source_file_name] to rename file Made@HackRU
